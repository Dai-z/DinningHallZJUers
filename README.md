# 浙大食堂食物中毒事件
梳理浙大食堂食物中毒事件中的相关证据与事件进展

## 摘要

10月23日晚，陆续有同学发现肠胃不适去校医院就诊，被诊断为肠道感染。校医院于10月24日10:00公告相关状况，公告有28人感染。该数字不被同学所认可，并陆续传出学校公关部门删帖、控制校医院感染诊断人数的消息。截至目前事件仍在调查。
